# HomeScan Pro 🛒
**Projeto de Engenharia de Software & Bootcamp II** 

**Desenvolvedora: Jordana Moreira Lopes** 

**Professor: Dr. Romes Heriberto** 

O **HomeScan Pro** é uma solução completa para gestão de listas de compras, desenvolvida para unir praticidade no cotidiano com conceitos avançados de engenharia de software, como automação de testes, integração contínua e consumo de APIs externas.


---

## 🚀 Link do Deploy:
[https://homescan-pro.vercel.app](https://homescan-pro.vercel.app)

## 🛠️ Stack Tecnológica
- **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+).
- **Estilização:** Tailwind CSS (via CDN) e CSS customizado.
- **Engenharia de Qualidade:** Jest (Testes de Integração).
- **DevOps:** GitHub Actions (CI/CD) e Vercel (Hospedagem).
- **Serviços Externos:** Quotable API (Frases) e MyMemory API (Tradução).
- **APIs:** [Quotable API](https://github.com/lukePeavey/quotable) e [MyMemory API](https://mymemory.translated.net/).


## 🚀 Funcionalidades Principais
1.  **Gestão Dinâmica:** Adição e remoção de itens com interface responsiva.
2.  **Tradução em Tempo Real:** Busca frases motivacionais em inglês e traduz automaticamente para o português.
3.  **Temas Personalizados:** Alternância entre Dark e Light Mode com persistência de dados.
4.  **Histórico Inteligente:** Salva e restaura listas finalizadas através do LocalStorage.
5.  **Qualidade Garantida:** Pipeline de CI que executa testes a cada commit.
